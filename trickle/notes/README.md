# ANAS ZOUZOU - Professional Portfolio

## Project Overview
A modern, responsive portfolio website showcasing the skills, experience, and contact information of Anas Zouzou, a Computer Science student specializing in programming and cybersecurity.

## Key Features
- **Responsive Design**: Fully responsive layout that works on all devices
- **Modern UI**: Clean, professional design with smooth animations
- **Navigation**: Sticky navigation with smooth scrolling
- **Printable CV**: Separate CV page optimized for printing and PDF export
- **Sections**: 
  - About Me
  - Skills (Programming, Cybersecurity, Design, Soft Skills)
  - Experience
  - Languages
  - Contact Information

## Technology Stack
- React 18
- TailwindCSS
- Lucide Icons

## Color Scheme
- Primary: #003366 (Navy Blue)
- Secondary: #0056b3 (Blue)
- Accent: #cce7ff (Light Blue)
